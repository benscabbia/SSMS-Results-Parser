SSMS Results Parser

This parser was developed to quickly compare two SQL queries. The parser will compare IO and Time data, but future builds will hopefully include data comparison.

Inputs

The parser in its current state is strict, and therefore requires the input to follow a specific format. More information here

Enable Results to Text

In SSMS, navigate to: Tools > Options > Query Results. Select Results to Text

We now need to enable IO and Time data in results. We can enable this information in two ways:

In SSMS, navigate to: Tools > Options > Query Execution > SQL Server > Advanced.

Enable SET STATISTICS TIME and SET STATISTICS IO

Alternatively at the top of your SQL Script you can append:

SET STATISTICS IO ON
SET STATISTICS Time ON
